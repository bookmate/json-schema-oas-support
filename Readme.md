Simple extension for supporting OAS in json-schema gem

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'json-schema-oas-support'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install json-schema-oas-support


## Tests

Just run:

    $ rspec spec
